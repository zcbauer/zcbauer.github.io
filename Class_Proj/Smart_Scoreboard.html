Background

This project was my second capstone project, done for EECS 473: Advanced Embedded systems. The class was split up into 5 person teams to work on a creating a product that was supposed to be commercial grade. As part of the requirments, each groups product needed to have a custom designed Printed Circuit Board onto which a microprocesser needed to be mounted. These requirments ensured that each groups PCB was sufficently complicated.

Product

My group decided on creating a scoreboard which allows the user to keep score in group gaming settings. We wanted to make it wirelessly controllable from multiple users, and we wanted it to be visible from a distance and to have a large enough internal battery to be able to last multiple hours unplugged.
